## 📖 About the project

This project was developed as part of my Front-End learning journey.

It simulates the website of a restaurant, including a login interface, user registration, and a digital menu with a clean and responsive design.

The objective of this project was to strengthen my skills in responsive web design, page structure, styling, and JavaScript interactions.
